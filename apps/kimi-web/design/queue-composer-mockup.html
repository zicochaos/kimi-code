这套设计如何回应之前的几个问题

  • 误触中断 → Stop 与 Send 物理分离，Send 永不变成 Stop。
  • 队列看不见 → 直接出现在对话流末尾，半透明 + 虚线边框表达“尚未发送”，不再需要点开 Dock 面板。
  • 不能重排 / 图片不可见 → ⋮⋮ 拖拽排序；带图片的项显示缩略图（示例里的 图片 ×2 可换为真实缩略图）。
  • 出队不可见 → 点“▶ 出队一条”看效果：队头气泡上浮为正式用户气泡，队列整体前移、“下一条”标签易主。
  • Steer 退居二级 → 保留 Ctrl+S，在输入框 hint 里提示，不抢主路径。
